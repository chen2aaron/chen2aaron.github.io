Use ZeroNet With Tor

ZeroNet 本身并不是匿名网络。但他可以与 Tor Browser 结合，实现匿名网络的功能。

  1. 安装 ZeroNet 和 Tor Browser，详细教程可以参见官网。

  2. 打开洋葱浏览器（Tor borwser）

    • 打开网址 about:preferences
    • 点击 网络代理 - 设置
    • 输入 127.0.0.1 并把这个地址加入到 不通过代理访问

  1. 编辑 Tor 的配置文件 torrc，添加一行，然后重启 Tor 服务

  2. 运行 ZeroNet 并打开 http://127.0.0.1:43110/Config，设置 Network - Tor - Always

  3. 找到 ZeroNet 的配置文件 zeronet.conf，增加 2 行 Tor 代理配置，保存并重启 ZeroNet 服务

  1. 用 Tor Browser 访问 http://127.0.0.1:43110 就可以啦。

Reference

  • ZeroNet中文安装指南 https://stgapr.github.io/zeronet-Documentation-ZH_CN/faq/
  • Tor 配置文件路径 https://www.torproject.org/docs/faq.html.en#AdvancedTorUsage
